
## c8ylp plugin

```
Usage: c8ylp plugin COMMAND [ARGS]...

  Run a custom plugin (installed under ~/.c8ylp/plugins/)

  Example 1:

      c8ylp plugin copyto device01 <src> <dst>

Options:
  -h, --help  Show this message and exit.

Commands:
  command     Start once-off proxy and execute a (local) script/command
```
